# 茶饮为什么开始“早餐化” — 素材与来源记录

## 主题判断
- 站内已存在：花香茶饮、轻乳茶健康叙事、补水感/电解质叙事、品牌扩张观察。
- 本轮新题区分点：不再围绕“花香”“低糖”“补水”或单一品牌，而是围绕 **工作日早晨时段、早餐化、饱腹感、通勤代餐想象** 的消费结构变化。

## 候选方向对比（本轮）
1. 早餐化 / 代餐感茶饮
   - 区分度：高
   - 深写潜力：高
   - 视觉潜力：高
   - 结论：采用
2. 果蔬茶 / 羽衣甘蓝茶饮
   - 区分度：中高
   - 但公开中文信号抓取不足，容易写成趋势速写
3. 东方冰茶热度延续
   - 与站内补水感、电解质叙事邻近，重复风险偏高
4. 牛油果/谷物茶饮回潮
   - 可写，但热度信号不够稳定
5. 午餐代餐茶饮
   - 和早餐化高度邻近，且更窄，最终并入主文判断
